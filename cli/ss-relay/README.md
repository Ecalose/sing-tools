# ss-relay

## Requirements

```
* Go 1.18
```

## Install

```shell
git clone https://github.com/SagerNet/sing-tools sing
cd sing

cli/ss-relay/install.sh
cli/ss-relay/enable.sh
```

## Log

```shell
sudo journalctl -u ss-relay --output cat -f
```

## Update

```shell
cli/ss-relay/update.sh
```

## Uninstall

```shell
cli/ss-relay/uninstall.sh
```
